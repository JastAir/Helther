# Helther
Server Helth Checker with WebAssembly web view (Writed on .NET Core 6, C#)
- Server: .NET Web API
  -  IHostedService for get added service and then ping
  -  SQLite for storage date
- FrontEnd: Blazor WebAssembly
  - Bootstrap
  - Updated every 5 sec
